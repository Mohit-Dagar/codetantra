# CodeTantra Solutions

A collection of solutions for **CodeTantra** platform programming challenges and lab exercises.

## About CodeTantra

[CodeTantra](https://codetantra.com) is a cloud-based interactive Teaching & Learning platform founded in 2014. It is widely used by universities and colleges across India for:

- **Coding Courses**: C, C++, Java, Python, Data Structures, Web Technologies
- **Online Assessments**: Auto-graded coding tests with hidden test cases
- **Lab Assignments**: Problem-solving exercises across various CS subjects
- **Proctored Exams**: AI-powered remote proctoring for online exams
- **Competitive Programming**: Algorithm and data structure challenges

The platform follows curricula from universities like AKTU and covers subjects aligned with standard engineering syllabi.

## Repository Purpose

This repository serves as a reference for CodeTantra coding solutions. It contains solved problems from various labs and programming exercises, helping students understand concepts and improve their problem-solving skills.

## Topics Covered

- C Programming (Basics, Operators, Control Flow, Functions, Pointers, Arrays, Strings)

## How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Mohit-Dagar/codetantra.git
   cd codetantra
   ```

2. **Browse solutions by topic** — Navigate to the relevant folder (e.g., `C/`, `Python/`) to find solutions for specific problems.

3. **Run the code** — Each solution file is self-contained and can be compiled/run using standard tools:
   - **C/C++**: `gcc filename.c -o output` or `g++ filename.cpp -o output`
   - **Python**: `python filename.py`
